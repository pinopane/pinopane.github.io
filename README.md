# Portfolio Daniel Pino

Landing one-page construida con React + Tailwind + Vite.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Ediciones rápidas

- Textos, experiencia, proyectos, skills y links: `src/content.js`
- Estructura visual por secciones: `src/App.jsx`
- Colores, tipografías y sombras del tema: `tailwind.config.cjs`
- Estilos globales y animaciones reveal/parallax suave: `src/index.css`
- SEO básico y fuentes: `index.html`
- Logo mostrado en navbar: `public/logo.jpg` o reemplaza `logo.jpg` en la raíz y actualiza la ruta en `src/App.jsx`
